# Front-End Web Project 🌐

Welcome to my very first university web development project! This is a responsive website built from scratch during my first year.

The primary goal of this project was to master the fundamentals of layout design, structuring content, and styling.

---

## 📌 Project Overview & Stack

This project is fully focused on **static front-end design**, strictly utilising core styling and structural markup. 

### Tech Stack Used:
* **HTML5:** For creating a clean, semantic web structure.
* **CSS3:** For custom styling, layouts, margins, and visual presentation.

> ⚠️ **Note on Functionality:** As this was an introductory first-year assignment, **no JavaScript** was used in this project. It is entirely a static showcase of HTML and CSS styling concepts.
